This form is designed to test the RainbowBrowserAI smart_actions tool. The tool can:

  • Automatically detect field types (name, email, phone, address, etc.)
  • Suggest appropriate values for each field type
  • Handle complex form structures with validation
  • Fill forms intelligently based on context and user preferences
Test Commands:
# Basic form filling with smart suggestions
rainbow-poc smart-actions --url file://$(pwd)/examples/smart_form_demo.html --smart-suggestions

# Fill form with specific data
rainbow-poc smart-actions \
  --url file://$(pwd)/examples/smart_form_demo.html \
  --form-data "firstname=John,lastname=Doe,email=john.doe@example.com" \
  --auto-submit

# Use custom preferences
rainbow-poc smart-actions \
  --url file://$(pwd)/examples/smart_form_demo.html \
  --user-preferences "firstname=Jane,lastname=Smith,city=San Francisco" \
  --intelligent-detection
